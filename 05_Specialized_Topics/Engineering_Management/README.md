# Engineering Management (工程管理與軟實力)

資深工程師 (Senior Engineer) 與技術主管 (Tech Lead) 不僅需要卓越的技術能力，更需要具備領導力、專案管理能力以及培養團隊的軟實力。本章節涵蓋了從技術領導、人才招聘到事故管理的關鍵主題。

## 學習建議

1. **角色轉變**: 理解從 Individual Contributor (IC) 到 Tech Lead 的心態轉變，重點在於「透過他人達成目標」。
2. **溝通為王**: 軟實力的核心是溝通。學習如何給予建設性回饋 (Feedback) 和進行有效的技術談判。
3. **系統化思考**: 將管理視為系統設計的一部分，優化流程 (Process) 以提升團隊效率。
