## 學習建議

1. **先廣度後深度**: 先閱讀 **技能圖譜**，了解 AI Engineer 的能力邊界。
2. **動手實作**: 申請一個 OpenAI API Key，嘗試寫一個簡單的 Chatbot 或 RAG 應用。
3. **關注生態系**: 熟悉 LangChain, LlamaIndex 等框架，以及 Vector Database 的使用。
