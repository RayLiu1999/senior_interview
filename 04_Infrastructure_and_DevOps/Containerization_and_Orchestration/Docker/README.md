# Docker

歡迎來到 Docker 面試問題集！

Docker 是一種開源的容器化平台，它徹底改變了軟體開發、部署和維運的方式。透過將應用程式及其依賴項打包到一個輕量級、可移植的容器中，Docker 實現了「一次建置，到處執行」的理念。

本章節涵蓋了 Docker 的核心概念、常用命令、底層原理以及在實際場景中的應用。
