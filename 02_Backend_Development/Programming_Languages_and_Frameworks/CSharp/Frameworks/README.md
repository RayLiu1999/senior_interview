# Frameworks（框架）

本節涵蓋 C#/.NET 生態系統中最重要的框架，包括 ASP.NET Core（Web 開發框架）和 Entity Framework Core（ORM 框架）。這些框架是建構現代 .NET 後端應用的基石。

## 子目錄

- [ASP.NET Core](./ASP.NET_Core/README.md) - 跨平台的高效能 Web 框架
- [Entity Framework Core](./EF_Core/README.md) - 現代化的物件關聯對映框架

## 學習路徑建議

### 初學者

1. 先掌握 C# 核心語法和非同步程式設計
2. 學習 ASP.NET Core 基礎（DI、中介軟體、路由）
3. 了解 Entity Framework Core 基本 CRUD 操作

### 進階

1. 深入 ASP.NET Core 請求管線和中介軟體原理
2. 掌握 EF Core 效能優化和進階查詢技巧
3. 學習認證授權機制（JWT、Identity）

### 專家

1. 自訂中介軟體和篩選器
2. EF Core 變更追蹤機制和併發控制
3. 微服務架構與 gRPC 整合
