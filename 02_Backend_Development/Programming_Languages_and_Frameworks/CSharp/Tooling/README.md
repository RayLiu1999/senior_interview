# Tooling（工具鏈）

本節涵蓋 C#/.NET 開發中常用的工具和配置，包括套件管理、CLI 工具、專案結構等。

## 主題列表

| 編號 | 主題 | 難度 | 重要性 | 標籤 |
| :---: | :--- | :---: | :---: | :--- |
| 1 | NuGet 套件管理 | 4 | 4 | `NuGet`, `Package` |
| 2 | dotnet CLI 使用 | 4 | 4 | `CLI`, `dotnet` |
| 3 | MSBuild 與專案結構 | 5 | 3 | `MSBuild`, `csproj` |
| 4 | 程式碼分析工具 | 5 | 3 | `Analyzer`, `Code Quality` |

---

## 學習建議

1. **熟悉 dotnet CLI**：dotnet new、dotnet build、dotnet run 等常用指令
2. **理解 NuGet**：套件版本管理、私有套件來源配置
3. **了解專案檔結構**：csproj 中的常用配置項目
4. **啟用程式碼分析**：使用 .NET Analyzers 提升程式碼品質
