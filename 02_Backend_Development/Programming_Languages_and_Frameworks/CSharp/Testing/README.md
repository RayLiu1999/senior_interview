# Testing（測試）

本節涵蓋 C#/.NET 中的測試框架、模擬工具和測試策略，是確保程式碼品質的重要環節。

## 主題列表

| 編號 | 主題 | 難度 | 重要性 | 標籤 |
| :---: | :--- | :---: | :---: | :--- |
| 1 | xUnit 與 NUnit | 4 | 4 | `xUnit`, `NUnit`, `Unit Test` |
| 2 | Moq 模擬框架 | 5 | 4 | `Moq`, `Mock`, `Stub` |
| 3 | 整合測試策略 | 6 | 4 | `Integration Test`, `WebApplicationFactory` |
| 4 | BenchmarkDotNet | 5 | 3 | `Benchmark`, `Performance` |

---

## 學習建議

1. **掌握 xUnit**：理解 Fact、Theory、Fixture 等核心概念
2. **熟練使用 Moq**：學會模擬依賴、驗證行為
3. **整合測試**：使用 WebApplicationFactory 進行 API 整合測試
4. **效能測試**：使用 BenchmarkDotNet 進行精確的效能量測
